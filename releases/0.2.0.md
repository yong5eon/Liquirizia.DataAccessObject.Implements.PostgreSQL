Table 과 View 에서 스키마를 정의해서 사용할 수 있음

## 예제

```python
from Liquirizia.DataAccessObject.Implements.PostgreSQL import Table, View


class SampleTable(
  Table,
  table='SampleTable',
  schema='SampleSchema',
):
  ...

class SampleView(
  View,
  executor=Select(SampleTable),
  view='SampleView',
  schema='SampleSchema',
):
  ...
```
